# La-r-serve-landing

Page de présentation de **La Réserve** — app de vigilance citoyenne · pilote Congo.

- `index.html` : page unique et autonome (polices, icônes 3D et téléphone animé embarqués en base64).
- Déployée sur **Vercel** (Root Directory = `./`, aucun build).
- L'**APK Android** est distribué via les [Releases](../../releases/latest) — le fichier
  (~190 Mo) dépasse la limite Git de GitHub (100 Mo), les Releases acceptent jusqu'à 2 Go.

Source de génération : projet principal `La réserve/landing/`.
